# 🛡️ AML Fraud Detection & Investigation Dashboard

An end-to-end Anti-Money Laundering (AML) analytics project designed to identify suspicious financial transactions using SQL, Python, Machine Learning, Excel, and Power BI. The project analyzes customer transactions, predicts suspicious activity, detects anomalies, and provides an interactive dashboard for financial risk monitoring.

---

## 🚀 Tech Stack

- Python (Pandas, NumPy, Matplotlib, Seaborn)
- MySQL Workbench
- Scikit-learn
- Microsoft Excel
- Power BI
